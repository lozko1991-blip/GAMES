⌨️ КЕРУВАННЯ НА ПК:

  • A/D / Стрілки◄ ► — Рух бійця вліво/вправо
  • Space / Стрілка ▲ — Стрибок (з A/D — Сальто вбік)
  • S / Стрілка ▼ — Присідання / Блок (назад від удару)
  • W / клавіша w — Аперкот (підбиває ворога в повітря)
  • S + D / S + A — Низька підсічка (збиває з ніг)
  • E / Num 4 / ЛКМ — Удар рукою (4 швидких натискання: Комбо-Ласо)
  • F / Num 5 / ПКМ — Удар ногою (4 швидких натискання: Супер-Ласо)
  • Shift — Кинути фігурки з неба (затиснути: Блискавка)
  • C / I / Num 0 / Коліщатко — Дістати / змінити зброю

⚔️ КЕРУВАННЯ ЗБРОЄЮ:

  • Коліщатко / Num 0 / Shift — Зміна зброї в руках
  • ЛКМ / E — Основна атака зброєю (постріл з лука)
  • ПКМ / F — Швидкий удар зблизька
